Pida Landing Page

Pida Landing Page was created on 2021 for PT Pabrikasi Interior Design Arsitektur, A company specializing in interior design.
Below is a brief overview of the Pida Landing Page Project.

In this project, I manually created a static web user interface using HTML, CSS, and Bootstrap, without relying on any templates.
The user interface is based on the design team's work, which I received as a Figma file and i converted it into a web-based interface.
The home page includes five sections: a header, portfolio showcase, testimonials, a 'reach us' form, and a footer.
